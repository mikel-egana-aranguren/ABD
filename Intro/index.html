Modelo relacional (SQL): coleccion de tablas inter-relacionadas

MySQL, PostgreSQL, Oracle, ...

El modelo relacional surgio en los 70 y desde entonces es el modelo mas usado

En los 70 las BBDD, estaban, por ejempo, en grandes instituciones:

  • Habia pocas
  • En entornos muy controlados
  • Cambiaban poco
  • No habia que integrarlas con otras DBs

En este curso

  • Bases de Datos de parejas de valores y wide column: Redis,
  • Bases de Datos orientadas documentos: MongoDB
  • Bases de Datos para Property Graphs: Neo4J
  • Bases de Datos para RDF (Triple Stores): GraphDB
  • Vectores: Chroma
